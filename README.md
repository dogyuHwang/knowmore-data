# knowmore-data

**아만보** 앱의 카드 콘텐츠 데이터 저장소입니다.

이 레포의 JSON 파일을 수정하면 앱 업데이트 없이 **최대 1시간 내** 사용자에게 반영됩니다.

---

## 주제별 카드

| 파일 | 주제 | 설명 |
|------|------|------|
| `hot_issues.json` | 이슈 | 매주 업데이트되는 최신 이슈 카드 |
| `economy.json` | 경제 | 경제 기본 개념과 시사 용어 |
| `stock.json` | 주식 | 주식 투자 관련 용어 |
| `ai.json` | AI | 인공지능 기술과 트렌드 |
| `it.json` | IT | IT 기술 용어 |
| `law.json` | 법률 | 법률 상식과 제도 |
| `health.json` | 건강 | 건강·의학 상식 |
| `real_estate.json` | 부동산 | 부동산 용어와 제도 |
| `glossary.json` | 용어사전 | 카드 내 어려운 용어의 인라인 설명 |

---

## 카드 구조

```json
{
  "id": "ai_001",
  "topic": "AI",
  "word": "인공지능",
  "summary": "인간의 지능을 모방해 학습·판단·예측할 수 있도록 만든 컴퓨터 시스템",
  "description": "인공지능(AI)은 컴퓨터가 스스로 학습하고 문제를 해결할 수 있도록 만든 기술이에요.",
  "example": "유튜브가 내 취향에 맞는 영상을 추천해주는 것도 인공지능 알고리즘 덕분이에요.",
  "level": "뉴비",
  "active": true
}
```

| 필드 | 설명 |
|------|------|
| `id` | 고유 식별자 (`{topic}_{번호}`) |
| `topic` | 주제 이름 |
| `word` | 카드 제목 (10자 이내) |
| `summary` | 한 줄 요약 (40자 이내) |
| `description` | 본문 설명 (3~5문장) |
| `example` | 실생활 예시 |
| `level` | 난이도: `뉴비` / `고인물` / `신` |
| `active` | 앱 노출 여부 (기본 `true`) |
| `updatedAt` | 이슈 카드 전용 날짜 |

---

## 난이도 시스템

앱은 뉴비를 전부 학습해야 고인물이 해금되고, 고인물을 전부 학습해야 신이 해금됩니다.

| 난이도 | 해금 조건 | 예시 |
|--------|----------|------|
| **뉴비** | 처음부터 | 인플레이션, GPT, 전세 |
| **고인물** | 뉴비 완료 후 | 테이퍼링, RAG, 역전세 |
| **신** | 고인물 완료 후 | 양적완화 출구전략, RLHF |

## 용어사전 (glossary.json)

카드 설명 텍스트에서 어려운 용어를 탭하면 쉬운 설명이 나타나는 인라인 사전입니다.

```json
{
  "terms": {
    "양적완화": "중앙은행이 돈을 찍어서 시장에 푸는 정책",
    "ETF": "여러 주식을 한 바구니에 담아 거래하는 펀드"
  }
}
```

---

## 퀴즈

앱은 객관식, 주관식, 유사어 구분 퀴즈를 자동 생성합니다.
유사어 구분 퀴즈는 카드에 `quiz.confusables` 필드를 추가하면 활성화됩니다.

```json
{
  "quiz": {
    "confusables": ["흐르무즈 해협", "호브루즈 해협"]
  }
}
```

---

## 스크랩

사용자가 카드를 스크랩하면 서버에서 삭제되어도 기기에서 계속 열람 가능합니다.
이슈 카드는 매주 교체되므로, 보관하고 싶은 카드를 스크랩할 수 있습니다.
